Template Literals

Usando Template Literals, ahora podemos construir strings que tienen caracteres especiales en ellas sin necesidad de escaparlas explícitamente.

var text = "This string contains \"double quotes\" which are escaped.";    

 let text = `This string contains "double quotes" which don't need to be escaped anymore.`;

Los Template Literals también admiten interpolación, lo que permite la tarea de concatenar strings y valores:

var name = 'Tiger';
var age = 13;

console.log('My cat is named ' + name + ' and is ' + age + ' years old.');

Mucho más simple que lo anterior se puede tener lo siguiente:

const name = 'Tiger';
const age = 13;

console.log(`My cat is named ${name} and is ${age} years old.`);

En ES6 se manejan nuevas lineas de la siguiente manera:

var text = (
    'cat\n' +
    'dog\n' +
    'nickelodeon'
);

O también así:

var text = [
    'cat',
    'dog',
    'nickelodeon'
].join('\n');

Los Template Literals conservarán las nuevas líneas sin tener que expresarlas explicitamente: 

let text = ( `cat
dog
nickelodeon`
);

los Template Literals también pueden aceptar expresiones como la siguiente: 

let today = new Date();
let text = `The time and date is ${today.toLocaleString()}`;
