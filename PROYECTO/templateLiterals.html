Template Literals

Usando Template Literals, ahora podemos construir cadenas que tienen caracteres especiales en ellas sin necesidad de escaparlas explícitamente.

var text = "This string contains \"double quotes\" which are escaped.";    

 let text = `This string contains "double quotes" which don't need to be escaped anymore.`;

Los Template Literals también admiten interpolación, lo que hace que la tarea de concatenar cadenas y valores:

var name = 'Tiger';
var age = 13;

console.log('My cat is named ' + name + ' and is ' + age + ' years old.');
