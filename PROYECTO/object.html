Destructuring Objects

Destructuring

Destructuring o en español desestructuración permite extraer valores de arrays y objetos, incluso si se encuentran profundamente anidados. Los valores obtenidos pueden ser almacenacenados en variables con una sintaxis conveniente.

Un ejemplo de Destructuring de Objects es el siguiente: 

var luke = { occupation: 'jedi', father: 'anakin' };
var occupation = luke.occupation; // 'jedi'
var father = luke.father; // 'anakin'

let luke = { occupation: 'jedi', father: 'anakin' };
let {occupation, father} = luke;

console.log(occupation); // 'jedi'
console.log(father); // 'anakin'
