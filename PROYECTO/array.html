Destructuring Arrays

Destructuring

Destructuring o en español desestructuración permite extraer valores de arrays y objetos, incluso si se encuentran profundamente anidados. Los valores obtenidos pueden ser almacenacenados en variables con una sintaxis conveniente.

Un ejemplo de Destructuring de Arrays es el siguiente: 

var arr = [1, 2, 3, 4];
var a = arr[0];
var b = arr[1];
var c = arr[2];
var d = arr[3];

let [a, b, c, d] = [1, 2, 3, 4];

console.log(a); // 1
console.log(b); // 2
