Spread Operator

Spread Operator u operador de propagación en español.
En ES5, se puede encontrar el máximo de valores en una matriz usando el método apply en Math.max como esto:

Math.max.apply(null, [-1, 100, 9001, -32]); // 9001

En ES6, ahora se puede utilizar el operador de propagación para pasar una matriz de valores para ser utilizados como parámetros de una función:

Math.max(...[-1, 100, 9001, -32]); // 9001

Se puede concatenar literales de arrays fácilmente con esta sintaxis intuitiva: 

let cities = ['San Francisco', 'Los Angeles'];
let places = ['Miami', ...cities, 'Chicago']; // ['Miami', 'San Francisco', 'Los Angeles', 'Chicago']
