Remplazo de los IIFEs por Blocks

Un uso común de Immediately Invoked Function Expressions es incluir valores dentro de su ámbito de aplicación. En ES6, ahora tenemos la capacidad de crear ámbitos basados en bloques y por lo tanto no están limitados puramente a ámbito basado en funciones.

Ejemplo usando Immediately Invoked Function Expressions:

(function () {
    var food = 'Meow Mix';
}());

console.log(food); // Reference Error

Usando bloques en ES6: 

{
    let food = 'Meow Mix';
};

console.log(food); // Reference Error
