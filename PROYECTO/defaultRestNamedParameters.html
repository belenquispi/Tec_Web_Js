Parameters

En ES5, se tiene diferentes maneras de manejar funciones que necesitaban valores predeterminados, argumentos indefinidos y parámetros nombrados. Con ES6, se puede lograr todo esto y más utilizando una sintaxis más concisa.

Default Parameters 

function addTwoNumbers(x, y) {
    x = x || 0;
    y = y || 0;
    return x + y;
}'

En ES6, se puede simplemente suministrar valores por defecto para parámetros en una función:

function addTwoNumbers(x=0, y=0) {
    return x + y;
}

addTwoNumbers(2, 4); // 6
addTwoNumbers(2); // 2
addTwoNumbers(); // 0

Rest Parameters

En ES5, se maneja un número indefinido de argumentos como:

function logArguments() {
    for (var i=0; i < arguments.length; i++) {
        console.log(arguments[i]);
    }
}

Utilizando el operador rest, se puede pasar una cantidad indefinida de argumentos:

function logArguments(...args) {
    for (let arg of args) {
        console.log(arg);
    }
}

Named Parameters

Uno de los patrones en ES5 para manejar los parámetros nombrados fue usar el patrón de objetos de opciones, adoptado desde jQuery.

function initializeCanvas(options) {
    var height = options.height || 600;
    var width  = options.width  || 400;
    var lineStroke = options.lineStroke || 'black';
}

Se puede lograr la misma funcionalidad usando la desestructuración como un parámetro formal para una función:

function initializeCanvas(
    { height=600, width=400, lineStroke='black'}) {
        // Use variables height, width, lineStroke here
    }

Si se quiere que todo el valor sea opcional, se lo puede hacerlo desestructurando un objeto vacío:

function initializeCanvas(
    { height=600, width=400, lineStroke='black'} = {}) {
        // ...
    }
