Strings

Con ES6, la biblioteca estándar ha crecido enormemente. Junto con estos cambios están los nuevos métodos que se pueden utilizar en strings, como: .includes() y .repeat().

.includes()

var string = 'food';
var substring = 'foo';

console.log(string.indexOf(substring) > -1);

En lugar de buscar un valor de retorno > -1 para denotar contención de string, podemos simplemente usar .includes () que devolverá un booleano:

const string = 'food';
const substring = 'foo';

console.log(string.includes(substring)); // true

.repeat()

function repeat(string, count) {
    var strings = [];
    while(strings.length < count) {
        strings.push(string);
    }
    return strings.join('');
}

En ES6 ahora tenemos acceso a una implementación terser: 

// String.repeat(numberOfRepetitions)
'meow'.repeat(3); // 'meowmeowmeow'
